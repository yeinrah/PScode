# PScode

-   알고리즘 문제풀이 코드들을 백업한 레포지토리입니다.<br>

-   사용하는 사이트는 백준, 코드업 등 입니다. 사이트별 폴더로 분리했습니다.<br>

-   코드의 제목은 사이트의 이름 및 문제번호입니다.<br>
    -   백준 : 클래스 이름 Main으로 변경
    -   코드업 : 클래스 이름 Main으로 변경
    -   SWEA : 클래스 이름 Solution으로 변경
